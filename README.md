1. undefined হল যে ভ্যারিয়েবলটা ডিক্লেয়ার করা হয়েছে কিন্তু তার কোনো ভ্যালু দেওয়া হয়নি যেমন:
   let a;
   console.log(a);
   এখানে কনসোল লগ করলে undefined আসবে

   আর null হল যেটাতে ইচ্ছা করে খালি ভ্যালু সেট করা হয়
   যেমম
   let b = null;

2.map() এর মাধ্যমে নতুন array তৈরি করে প্রতিটি element পরিবর্তন করে return দেয়া হয়
যেমন:
let nums = [1,2,3];
let result =nums.map(n => n\*2);
এখানে result কে কনসোল লগ করলে [2,4,6] - এই অ্যারে পাওয়া যাবে।

এবং forEach() হচ্ছে যেটা শুধু loop চালায় নতুন array return করে না।যেমন :
nums.forEach(n => console.log(n));

এখানে nums এর যতগুলো এলিমেন্ট আছে প্রতিটা এলিমেন্ট এর জন্য লুপ চলবে, কিন্তু এর মাধ্যমে নতুন অ্যারে তৈরি হবে না.

3. শুধু দুইটা সমান চিহ্ন == দিয়ে শুধু ভ্যালু compare করা হয় আর
   টাইপ convert করা হয়।

5 == "5" // true

এখানে int এবং string কে সমান বা একই বলা হচ্ছে যা ভুল।

অন্যদিকে === দিয়ে
ভ্যালু আর টাইপ দুটোই কঠোর ভাবপ compare করা হয় এক্ষেত্রে
5=== "5"
এটা false আসবে.

4.আমরা সার্ভার থেকে ডাটা আনলে সার্ভার থেকে ডাটা আসতে যে সময় লাগে সেই সময়ে JavaScript যেন আটকে না থাকে, সেজন্য async/await ব্যবহার করা হয়। এতে করে কোড সহজ ও পরিষ্কার হয় এবং promise .then() না লিখেও কাজ হয় এক্ষেত্রে ডাটা না আসা পর্যন্ত await অপেক্ষা করে

5.Scope এর মানে হলো কোড এর কোন জায়গা থেকে ভ্যারিয়েবল ব্যবহার করা যাবে।
৩ ধরনের scope আছে: প্রথম হল Global Scope যদি ভ্যারিয়েবল বাইরে তৈরি করা হয় এতে করে সব জায়গা থেকে ব্যবহার করা যায়।
দ্বিতীয় হল Function Scope এটা যদি ফাংশনের ভিতরে ভ্যারিয়েবল তৈরি করা হয় তাহলে শুধু ওই ফাংশনের ভিতরেই ব্যবহার করা যাবে।
তৃতীয় হল Block Scope যেটা { } এর ভিতরে let বা const দিয়ে ভ্যারিয়েবল বানালে তা শুধু ওই block এর ভিতরে থাকবে।
